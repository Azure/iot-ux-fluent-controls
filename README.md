## Build
1. git clone https://github.com/Azure/iot-react-controls.git
2. npm install
3. npm run build

## Docs
1. npm run docs:build
2. npm run docs
3. You can now view style guide in the browser: 
   
    - `On Local:`         http://localhost:6060/
   
    - `On your network:`  http://10.0.75.1:6060/
